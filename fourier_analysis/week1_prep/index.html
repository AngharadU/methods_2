These series are only truly valid when we include an infinite number of terms.

However, in practice, it is not possible to write down or plot infinite numbers of terms. Therefore, this is approximated by working out the first terms. Such approximations are known as partial sums (because they’re only part of the total sum required).

A series is said to be converging if it gets closer and closer to the original function as more and more terms are added.
